# Caffeinate

The built-in `caffeinate` command can keep your Mac from going to sleep whilst it is running.

# Usage

```sh
$ caffeinate
```

Use <key>Ctrl</key>+<key>C</key> to abort the command.

## Timed

Use `-t` to specify a timeout (in seconds)

```sh
$ caffeinate -t 3600
```
